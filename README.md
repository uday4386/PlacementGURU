# PlacementPro

Modern placement management portal for colleges and universities — students, coordinators, and admins in one dashboard.

## Features

- Landing page with placement overview
- Role-based dashboards (Admin, Coordinator, Student)
- Login portal with quick role access
- Charts, stats, and activity feeds

## Tech Stack

- React 18 + TypeScript
- Vite 6
- Tailwind CSS 4
- React Router
- Recharts
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```
